# MSc-Thesis---Amine-Alami
This repository contains the code of my MSc thesis project. It is structured as follows:

There are four main folders:

1. minimal_models --> Contains python files and notebooks that build the finite element models. 
2. models --> Contains python files with stiffness and mass matrices of relevant beam/rod models. 
3. scripts --> Contains python files in which relevant functions for the FEM implementation such as element.py and nodes.py. 
4. tests --> Contains sanity checks and tests of my code. 

The minimal model describes the dynamic behaviour of the Maeslantkering. This is a large storm surge barrier near Rotterdam. The minimal model is validated with a more extensive finite element model which was developed in FEM package ANSYS 2025 R2. 

Feel free to use these folders for academic purposes.
